## 项目简介

此项目是一个基于Django的旅游网站，主要用于展示红色旅游景点的路线。它允许管理员管理景点信息和分类，用户可以浏览景点列表、查看景点详情并评价景点。

## 主要功能

- 用户登录注册、退出系统
- 浏览景点列表，按分类过滤、模糊搜索
- 查看景点详情，包括路线信息
- 评价景点
- 提供反馈建议
- 修改个人信息


## 技术架构

- 前端：HTML、CSS、JavaScript
- 后端：Django


## 使用说明

1. 使用虚拟环境安装项目：
```
pip install -r requirements.txt
```

2. 运行项目：
```
python manage.py runserver
```

3. 使用浏览器访问项目地址：`localhost:8000`
